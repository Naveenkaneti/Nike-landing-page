# Premium Nike Shoe Landing Page

A high-fidelity, premium landing page for Nike shoes, inspired by the official Nike.in dashboard. This project features immersive animations, a functional cart system, and a Node.js backend, all built with a "Creative & Non-Basic" approach.

## 🚀 Live Demo
**Check out the live site here:** [https://nike-landing-page-nine-eta.vercel.app/](https://nike-landing-page-nine-eta.vercel.app/)

## ✨ Key Features
- **Authentic Nike UI**: Replicated the official Nike.in dashboard structure, including the top utility bar and main navigation categories (Men, Women, Kids, Sports, etc.).
- **Immersive GSAP Animations**:
  - **Dynamic Hero Intro**: High-performance zoom and rotation effects for the hero shoe.
  - **Floating Motion**: Infinite floating animation for a 3D-like product feel.
  - **Mouse-Follow Glow**: A creative glow effect that follows the user's cursor.
- **Interactive Cart System**:
  - Add items to your cart with real-time count updates and "pop" animations.
  - A slide-in **Cart Drawer** to view added items and calculate the total price.
- **Shop By Category**: Creative grid section with large, high-quality tiles for Men, Women, Kids, and Sports.
- **Official Redirects**: Clicking on categories or specific "Shop" buttons redirects you to the corresponding collection on the official Nike store.
- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop with a custom-built hamburger menu for smaller screens.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Animations**: [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform) and [AOS](https://michalsnik.github.io/aos/) (Animate On Scroll).
- **Backend**: Node.js & Express.js (for serving assets and API simulation).
- **Assets**: High-resolution, AI-generated transparent shoe images.
- **Deployment**: [Vercel](https://vercel.com/).

## 📖 How It Was Done
1. **Design First**: Focused on a dark/light hybrid aesthetic using radial gradients and mesh patterns to give it a premium feel.
2. **Animation Timeline**: Used GSAP timelines to orchestrate a sequenced entry where the background, shoe, and text appear in a choreographed flow.
3. **Cart Logic**: Implemented a state-based cart in JavaScript that manages item additions and dynamically updates the DOM for the drawer.
4. **Backend Integration**: Set up an Express server to handle routing and serve as a foundation for future e-commerce expansions.
5. **Vercel Optimization**: Configured `vercel.json` for serverless function support to host the Node.js backend seamlessly.
